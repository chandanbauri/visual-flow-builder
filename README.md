# Visual Flow Builder

A modern, node-based visual flow builder for designing complex logic and AI workflows.

## Features

- Dynamic node types: LLM, Data Source, Knowledge Transfer, Condition, Input, Output, Filter.
- Automatic graph layout using Dagre.
- Real-time JSON schema validation and preview.
- Intelligent start-node re-orientation.
- Premium glassmorphic UI with dark theme.

## Getting Started

1. Install dependencies:
   npm install

2. Run development server:
   npm run dev

3. Build for production:
   npm run build
